# 重力異常逆問題

重力異常逆問題は、地表またはその近くで観測された重力データから地下の密度分布を推定する、古典的な逆問題です。
多項式フィットとは異なり、典型的には **劣決定** の最小二乗問題になります。
そのため、`LSQSolver` の最小ノルム解は、この分野で重要な応用例です。

この例の目的は、最小ノルム解が常に地質学的に妥当なモデルであることを示すことではありません。劣決定逆問題では解が一意ではなく、最小ノルム解は一つのモデリング上の選択であることを示すことです。

この仮定が意図した事前情報と一致しない場合、復元されたモデルは期待される物理的・地質学的モデルと異なることがあります。したがって、ユーザーは自分の問題設定がどのような解を選んでいるのかを理解し、必要に応じて重み、正則化項、範囲制約、その他の事前情報を導入する必要があります。

---

## 問題設定

地下領域を直方体セル、すなわちボクセルに離散化します。次の記号を用います。

- $`\rho_j`$ : ボクセル $`j`$ の密度コントラスト
- $`g_i`$ : 観測点 $`i`$ で観測される重力異常

すべての観測値をまとめると、線形方程式

```math
A\rho = g
```

を得ます。ここで、$`A`$ は感度行列、$`\rho`$ は未知の密度コントラストベクトル、$`g`$ は観測された重力異常ベクトルです。

感度行列は、各ボクセルが各重力観測値にどのように寄与するかを表します。この文書では、$`\rho`$ は絶対密度ではなく **密度コントラスト** として扱います。たとえば背景密度を $`\rho_0`$ とすると、

```math
\rho = \rho_{\mathrm{rock}} - \rho_0.
```

合成例で用いる典型的な密度コントラストは、およそ $`100`$〜$`500\ \mathrm{kg/m^3}`$ です。

---

## 感度行列

観測点を $`(x_i,y_i,z_i)`$、ボクセルを $`V_j`$ とすると、鉛直方向の重力寄与は次のようにモデル化できます。

```math
A_{ij}
=
G
\iiint_{V_j}
\frac{z_i-z}
{\left[
(x_i-x)^2
+
(y_i-y)^2
+
(z_i-z)^2
\right]^{3/2}}
\,dV,
```

ここで、

- $`G`$ : 万有引力定数
- $`(x_i,y_i,z_i)`$ : 観測点
- $`V_j`$ : 未知量 $`\rho_j`$ に対応するボクセル

です。

この積分は直方体プリズムに解析解を持ちますが、単純な数値例ではボクセル中心でカーネルを評価し、体積を掛けることで近似できます。

```math
A_{ij}
\approx
G |V_j|
\frac{z_i-z_j}
{\left[
(x_i-x_j)^2
+
(y_i-y_j)^2
+
(z_i-z_j)^2
\right]^{3/2}},
```

ここで $`(x_j,y_j,z_j)`$ はボクセル $`j`$ の中心です。係数 $`G`$ は感度行列に含めることも、後で重力異常に掛けて物理単位に変換することもできます。たとえば、$`G`$ を省いた行列で計算した異常を $`\mathrm{m/s^2}`$ に直すには、最終的に $`G`$ を掛ければよいです。mGal で表示する場合は、$`1\ \mathrm{mGal}=10^{-5}\ \mathrm{m/s^2}`$ を用います。

```math
1\ \mathrm{mGal} = 10^{-5}\ \mathrm{m/s^2}.
```

## なぜ劣決定問題になるのか？

典型的な重力異常逆問題では、未知の密度セル数が観測点数より多くなります。たとえば、$`32 \times 32`$ の観測格子と $`32 \times 32 \times 4`$ の地下メッシュでは、

```math
m = 1024,\qquad n = 4096,
```

となり、

```math
A \in \mathbb{R}^{1024\times4096},\qquad m<n.
```

したがって、同じ重力観測値を説明できる密度モデルは多数存在します。`LSQSolver` は、その中から **最小ノルム解** を返します。これは、最小二乗解の中でユークリッドノルムが最も小さい解です。

これは便利な性質ですが、仮定を含まないわけではありません。適切な事前情報がなければ、劣決定逆問題から物理的に妥当なモデルが得られるとは限りません。

---

## アルゴリズムの概要

この例では、単純な合成逆問題の流れに従います。

1. 観測面と地下領域を離散化する。
2. 感度行列 $`A`$ を組み立てる。
3. 真の密度コントラストモデル $`\rho_{\mathrm{true}}`$ を定義し、対応する重力異常を計算する。

```math
g = A\rho_{\mathrm{true}}.
```

4. LSQSolver を用いて、劣決定逆問題

```math
A\rho \approx g
```

を解く。
5. 復元された最小ノルムモデル $`\rho_{\mathrm{est}}`$ と、設定したモデル $`\rho_{\mathrm{true}}`$ を比較する。

重要なのは、逆解析が元の合成モデルを直接復元しようとしているわけではなく、代数的な問題設定によって選ばれた最小ノルム解を計算している点です。

## 数値実験と結果

ここで示したいのは、最小ノルム解が代数的には明確であっても、必ずしもユーザーの意図した物理モデルではないことです。

### 実験 1: 順問題の確認

まず、地下中央付近の観測面に近い位置に正の密度コントラストを置き、対応する重力異常を計算します。これで、メッシュのインデックス、符号規約、観測点の位置、単位変換が整合していることを確認できます。

### 実験 2: 最小ノルム逆解析

次に、実験 1 で得られた合成重力異常を用いて逆問題を解きます。密度セル数が観測点数より多いため、同じ重力異常を説明できる密度モデルは多数存在します。LSQSolver は、その中から最小ノルム解を選びます。

ノイズのない合成例では、残差はほぼゼロになります。

```math
\|A\rho_{\mathrm{est}} - g\|_2 \approx 0.
```

しかし、復元されたモデル $`\rho_{\mathrm{est}}`$ が、設定したモデル $`\rho_{\mathrm{true}}`$ と一致するとは限りません。これはソルバーの失敗ではなく、劣決定逆問題の非一意性によるものです。

### 実験 3: 深さ方向の曖昧性

最後に、同じ水平位置にある密度ブロックをより深い位置に置いた場合を考えます。真の密度異常が深くなるほど、地表で観測される重力異常は滑らかで弱くなります。追加の事前情報なしに逆問題を解くと、復元された最小ノルムモデルは、真のモデルよりも観測面に近い側に異常を置くことがあります。

このことは、この例の主張をよく表しています。

> 最小ノルム解は数学的には明確ですが、必ずしもユーザーが意図した密度モデルではありません。

## 考察

重要なのは、LSQSolver がユーザーが与えた代数的な問題を解く点です。劣決定系に対しては、最小ノルム解を返します。これは事前情報がない場合の自然な選択ですが、解の物理的な意味を保証するものではありません。

したがって、重要な問いは次のようになります。

> 解はどのような事前情報を満たすべきか？

適切な事前情報を入れなければ、劣決定逆問題は観測値をよく再現しながら、物理的または地質学的に適切でない密度モデルを生成することがあります。たとえば、期待される深さ範囲、滑らかさ、密度コントラストの範囲、スパース性、既知の地質境界などが挙げられます。

このような仮定は、LSQSolver が自動的に決めるものではなく、ユーザーが問題設定を通して導入する必要があります。一般的には、重みや正則化項を用いて問題を再定式化する方法があります。

---

## まとめ

この例では、LSQSolver が重力逆解析に現れる密な劣決定最小二乗問題を解けることを示しました。

主な教訓は、系が解けることだけでなく、返される解が特定の意味を持つことです。

- 問題は劣決定である。
- 同じ重力データに適合する密度モデルは多数存在し得る。
- LSQSolver は最小ノルム解を返す。
- 最小ノルム解は、ユーザーの物理的な事前情報と一致するとは限らない。

したがって、LSQSolver を逆問題に適用する場合、意味のある解を得るために必要な事前情報を慎重に考える必要があります。単純な最小ノルム仮定が適切でない場合、その情報は重み、正則化項、制約、その他のモデル仮定として問題に反映する必要があります。

同じ考え方は、未知パラメータと観測値の関係が

```math
A x \approx b
```

として表される、多くの科学技術分野の逆問題にも適用できます。

## 付録: 正則化と事前情報

この節では、上の考察を補足する。正則化は、逆問題から仮定を取り除く方法ではなく、事前情報を最小二乗問題に組み込む方法である。代表例として Tikhonov 正則化がある。

```math
\min_{\rho}
\left(
\|A\rho-g\|_2^2
+
\lambda^2\|L\rho\|_2^2
\right),
```

ここで、$`L`$ は選択した事前モデルを表し、$`\lambda`$ はその強さを制御する。たとえば $`L=I`$ は大きな密度コントラストを抑え、差分作用素を使うと粗いまたは振動的なモデルを抑制できる。

この問題は、通常の最小二乗問題として書き直せる。

```math
\begin{bmatrix}
A \\
\lambda L
\end{bmatrix}
\rho
\approx
\begin{bmatrix}
g \\
0
\end{bmatrix}.
```

したがって、LSQSolver を使うには、

```math
\widehat{A} =
\begin{bmatrix}
A \\
\lambda L
\end{bmatrix},
\qquad
\widehat{g} =
\begin{bmatrix}
g \\
0
\end{bmatrix},
```

を定義し、$`\widehat{A}\rho \approx \widehat{g}`$ を解けばよい。重み付き最小ノルム定式化も同様に扱える。目的関数が $`\|W\rho\|_2`$ であれば、$`y=W\rho`$ とおいて $`A W^{-1} y \approx g`$ を解き、$`\rho=W^{-1}y`$ で元の変数を復元する。
